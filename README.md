# YAD Cloud

YAD Cloud is a personal, multi‑user file hosting solution built with Node.js, Express, and EJS.

## Features

- User authentication (session‑based)
- Upload multiple files (drag & drop or select)
- Owner‑only rename & delete
- Share files with other users (read‑only access)
- Search & sort (name, date, size)
- Download any selection as a single ZIP
- Change own password; admin can add new users
- Persistent storage using local JSON files

---

> **YAD Cloud** is a personal, multi‑user file hosting solution built with Node.js, Express, and EJS. It allows authenticated users to upload, share, rename, delete, and download files (individually or as a ZIP). All data is stored locally in a configurable directory, with user permissions and metadata kept in JSON files. Ideal for small teams or private use behind a reverse proxy.
